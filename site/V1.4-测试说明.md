# 山迹 V1.4 路线复核版测试说明

## 本版重点

- 新增 `review.html` 路线复核看板。
- 新增 `data/route-review-v1.4.csv`，按路线生成复核字段。
- 新增 `route-review-v1.4.md`，汇总每条路线的完成度、阻塞项和下一步。
- 路线详情页新增“复核状态”面板和 12 个检查项。
- 首页路线卡片和详情抽屉展示复核等级、复核完成度。

## 测试步骤

1. 打开首页，确认顶部显示“V1.4 路线复核版”。
2. 确认首页有“查看复核看板”入口。
3. 打开 `review.html`，确认显示 32 条候选路线、平均完成度和复核卡片。
4. 打开 `routes/FZ031.html`，确认有“复核状态”、12 个检查项、轨迹地图和“下载轨迹文件”按钮。
5. 打开 `data-check-v1.4.md`，确认问题数量为 0。

## 成功标志

- 首页路线数量为 32。
- 复核看板路线卡片数量为 32。
- FZ031 详情页复核完成度为 50 分。
- 浏览器控制台没有脚本错误。

## 当前结论

V1.4 已具备内测复核工作台雏形。当前 32 条路线都还处于“待补全”，主要原因是停车、厕所、补给、下撤点、地图点位和轨迹授权仍需人工确认。
