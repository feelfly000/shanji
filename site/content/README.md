# 山迹内容维护目录

这个目录用于后续把路线库从“演示数据”升级成“可长期维护的内容库”。

## 建议结构

- `routes/`：每条路线一份Markdown详情。
- `gpx/`：每条路线对应的GPX轨迹文件。
- `images/`：路线封面图、路口图、观景点图。

## 命名规则

建议所有内容都用路线ID命名：

```text
routes/FZ001.md
gpx/FZ001.gpx
images/FZ001-cover.jpg
images/FZ001-01.jpg
```

## 发布规则

- 候选路线可以有Markdown草稿，但不开放GPX下载。
- 已实地复核路线可以进入正式路线库。
- GPX轨迹必须完成清洗和风险标注后再开放下载。
