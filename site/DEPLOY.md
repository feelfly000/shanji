# 山迹静态部署说明

## 当前站点性质

山迹目前是纯静态网站，可以部署到：

- GitHub Pages
- Vercel
- Netlify
- 任意静态文件服务器

## 上线前需要确认

- 域名。
- 是否需要备案。
- 是否使用国内服务器。
- 真实路线是否完成复核。
- 测试 GPX 是否替换为真实轨迹。
- 活动报名链接是否为俱乐部官方链接。

## 部署目录

部署 `shanji-site/` 整个目录。

首页文件：

```text
index.html
```

## 公开测试建议

先不要直接大规模传播。建议先邀请：

- 3-5位徒步用户。
- 1-2个户外俱乐部。
- 1位熟悉福州路线的人。

收集反馈后再继续扩展路线复核、活动来源审核和正式域名发布。

## V1.1 当前状态

- 30 条候选路线。
- 5 条示例俱乐部活动。
- 支持路线表格和活动表格生成页面。
- 已有站点地图、robots、安全说明、关于页面、反馈入口。
- 正式上线前仍需替换 `https://example.com/` 为真实域名。
