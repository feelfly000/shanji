2. 点击“查看轨迹库”，确认能看到 3 条轨迹。
3. 打开 `routes/FZ031.html`，确认有轨迹预览图和轨迹下载入口。
4. 打开 `routes/FZ032.html`，确认有轨迹预览图和轨迹下载入口。
5. 打开 `data-check-v1.3.md`，确认问题数量为 0。
