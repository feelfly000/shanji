# 山迹 V0.6 交付清单

## 版本目标

V0.6 是公开测试准备版，重点补齐上线前的基础页面、SEO、站点地图和反馈入口。

- 首页升级为 V0.6 公开测试准备版。
- 增加 SEO 描述和 Open Graph 基础信息。
- 新增 `about.html` 关于山迹。
- 新增 `safety.html` 安全说明。
- 新增 `feedback.html` 反馈入口。

- 首页显示 V0.6。
