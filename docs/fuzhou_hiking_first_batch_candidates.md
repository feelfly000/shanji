# 福州徒步网站首批20条候选路线清单

说明：这份清单用于网站第一版的内容排期和样例录入，不等同于最终路况确认。距离、爬升、交通、开放状态和风险点需要后续实地复核或以官方最新公告为准。

## 选线原则

首批路线建议覆盖四种用户需求：

1. 新手和亲子：交通方便、强度低、下撤容易。
2. 市区半日：适合周末上午或傍晚完成。
3. 山海风景：适合传播和拍照。
4. 进阶路线：给有经验用户提供更高强度选择。
