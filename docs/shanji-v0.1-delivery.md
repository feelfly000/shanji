# 山迹 V0.1 内测版交付清单

## 版本定位

山迹 V0.1 是小范围内测版，用于验证路线查询、路线详情、活动聚合和安全提示的基础体验。

## 已完成

- 10条内测路线。
- 10份路线Markdown详情。
- 5条俱乐部活动样例。
- 自己选路线 / 本周俱乐部活动 双入口。
- 路线搜索。
- 区域、难度、主题、人群和距离筛选。
- 推荐、距离、用时、难度排序。
- 路线详情抽屉。
- 简化路线分布图。
- 俱乐部活动筛选。
- GPX下载入口说明。
- 内测安全说明和免责声明。
- V0.1测试清单。
- 可维护项目结构。

## 主要文件

- `shanji-site/index.html`：内测版首页。
- `shanji-site/assets/styles.css`：样式。
- `shanji-site/assets/app.js`：交互逻辑。
- `shanji-site/data/routes.js`：10条路线数据。
- `shanji-site/data/activities.js`：5条活动数据。
- `shanji-site/content/routes/`：10份路线Markdown详情。
- `shanji-site/V0.1-内测说明.md`：给测试者看的说明。
- `shanji-site/V0.1-测试清单.md`：验收和反馈清单。
