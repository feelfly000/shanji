## 建议工作流

1. 先打开 CSV，筛选 candidate_url 不为空的线路，逐张确认许可证和署名。
2. 对 candidate_url 为空的线路，优先打开 commons_search_url 和 openverse_search_url。
3. 普通图片搜索只用来找原作者或官方来源，不直接复制图片。
4. 如果图片不是线路实拍，上线时建议标注“示意图”。
